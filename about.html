About Me

My journey in cloud computing

Hello! I'm Mehdi Salhi, a computer science student who discovered a passion for cloud computing, particularly AWS.

What started as curiosity about cloud technologies has evolved into a focused journey of learning and building scalable solutions on the AWS platform. I'm fascinated by how cloud services enable developers to build and deploy applications faster and more reliably than ever before.

Currently, I'm actively expanding my AWS expertise and working towards obtaining AWS certifications. I'm diving deep into serverless architectures, infrastructure as code with Terraform, and AI application development, learning by building real-world projects like this portfolio.

What drives me in tech is simple: I love building things and solving problems. Cloud computing combines both perfectly—giving me the tools to architect solutions and the challenges to keep learning every day.
